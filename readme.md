# Foodie’s Life 🍔🍕

A responsive food delivery website built using **HTML** and **CSS**.
This project showcases a modern landing page for a restaurant/food ordering platform with attractive UI sections like Hero Banner, Popular Dishes, About Us, and Footer.

---

## 🚀 Features

* Responsive navigation bar
* Attractive hero section with background image
* Popular dishes showcase
* About Us section
* Clean and modern UI
* Hover effects on menu items
* Simple and beginner-friendly structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3
